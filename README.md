This portfolio project demonstrates a **Data Science and Machine Learning application** 
that analyses financial news sentiment and explores its relationship with stock market movements.

## Overview
The project combines textual sentiment analysis with daily market data to examine how news tone 
affects next-day price direction. It features:
- A **Logistic Regression model** trained on the *Financial PhraseBank dataset*.
- A merged dataset combining sentiment scores with market prices.
- An interactive **Streamlit dashboard** visualizing market and sentiment trends.

## Technologies
- Python, Pandas, NumPy  
- Scikit-Learn, Joblib  
- Matplotlib, Streamlit  
- YFinance (for market data)
